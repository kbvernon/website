## Academic Website Repository

This is the repository for my [academic website](https://www.kbvernon.io/). The 
design is extremely minimal and meant to provide a quick overview of academic 
output. The goal is to make it easier for me to keep this up-to-date by only 
having to add or edit entries in Zotero. 

The website itself is rendered using the R package 📦 [`extera`](https://github.com/kbvernon/extera),
an [extendr](https://github.com/extendr/extendr)-powered wrapper around the 
[tera](https://keats.github.io/tera/) templating engine in Rust. The cv is 
rendered using [typst](https://typst.app/) - may we never LaTeX again 🙏!
