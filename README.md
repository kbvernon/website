## Academic Website Repository

This is the repository for my [academic website](https://www.kbvernon.io/). The
website itself is rendered using the R package 📦 [`extera`](https://github.com/kbvernon/extera),
an [extendr](https://github.com/extendr/extendr)-powered wrapper around the tera 
templating engine in Rust. The cv is rendered using [typst](https://typst.app/) - 
may we never LaTeX again 🙏!
